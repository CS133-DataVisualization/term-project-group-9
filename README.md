
Package requirements: 
- pandas
- numpy
- matplotlib
- seaborn
- plotly
- warnings
- geopy
- panel
- hvplot
- sklearn

Instructions for running code: The project github repository should be cloned to a local machine and ran with a virtual environment such as python 3.12.7 if using an ide. Additionally, the ide should be capable of running jupyter notebooks. 
If not, and using a text editor such as visual studio code, the jupyter extension by microsoft can be installed to provide support.
Cells are ran in order.
